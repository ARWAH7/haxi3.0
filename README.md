# 波场 (TRON) 哈希实时分析大盘 2.0 实战增强版

本平台是一款专为波场区块链（TRON Network）深度分析师打造的实时数据可视化与趋势预测大盘。基于 React 19 + TypeScript + Tailwind CSS 构建，旨在提供军工级的数据稳定性与毫秒级的趋势响应。

---

## 💎 2.0 版本核心特性

### 1. 🧬 四维联动监控矩阵 (Quad-Core Monitoring)
系统支持对波场哈希的“单双”与“大小”进行全方位监控，核心组件包括：
- **动态走势图 (Big Road)**：采用经典大路算法，自动根据结果变化切列，支持智能滚动跟随。
- **强化珠盘路 (Bead Road)**：提供 1-20 行自定义高度设置，支持右侧行号精准对齐与单行长龙检测。
- **实时流水大盘 (Live Feed)**：每秒一次高频捕获 TronGrid 最新区块，支持哈希值穿透查询。

### 2. 🧠 AI 稳定演算引擎 v4.0 (AI Stable Engine)
集成了四种顶级统计学模型，仅在置信度 >92% 时输出信号，追求极端稳定性：
- **马尔可夫链 (Markov Chain)**：分析状态迁移概率，擅长捕捉震荡与规律性交替。
- **贝叶斯后验推理 (Bayesian Inference)**：通过大数定律监控分布偏差，寻找概率回归极点。
- **频谱周期律检测 (Spectral Analysis)**：利用傅里叶变换破译哈希流中隐藏的固定步长重复。
- **密集簇群共振 (Density Clustering)**：扫描微观窗口内的分布密度，锁定能量爆发点。

### 3. 🐉 全域长龙雷达 (Global Streak Radar)
针对 100+ 采样规则进行实时毫秒级扫描：
- **双模检测**：支持“走势长龙”与“珠盘行长龙”双维度提醒。
- **智能关注**：支持一键收藏核心关注点，AI 引擎将优先为您的关注项分配算力。
- **动态阈值**：支持批量设置长龙触发阈值，从 3 连到 15 连均可精准捕捉。

## 🛠️ 技术规格与架构

- **前端框架**：React 19 (Strict Mode)
- **样式引擎**：Tailwind CSS (响应式布局，完美适配移动端)
- **图标系统**：Lucide React
- **数据源**：TronGrid API (支持 TRON-PRO-API-KEY 鉴权)
- **本地持久化**：
  - `localStorage`：存储您的 API Key、自定义采样规则、个性化配色方案。
  - `Memory Cache`：内存级区块数据缓存，避免重复请求，极大降低 API 消耗。

## 📖 实战操作指南

### 第一步：配置核心引擎
在“设置”面板中输入您的 **TronGrid API Key**。
> *建议：使用具有付费配额的 Key 以支持每秒一次的高频轮询。*

### 第二步：定义采样规则
您可以根据实战需求新增或修改规则：
- **步长 (Interval)**：设置采样区块的间隔（如 1、20、60 区块）。
- **偏移 (Start Block)**：设置起始区块对齐高度。
- **UI 适配**：单独为每条规则设置走势图与珠盘路的显示行数。

### 第三步：多维趋势分析
- **综合盘面**：快速总览当前规则下的四项核心指标。
- **长龙榜单**：寻找当前全网最长龙的机会，点击卡片可实现“瞬时跳转”。
- **AI 预测**：参考高置信度信号，结合“风险等级”与“哈希熵值”进行决策。
